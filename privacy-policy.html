Privacy Policy

Last updated: 2026-07-15

History Talks Upload Tool ("the tool") is a private, single-user automation script built and operated by its developer to publish videos to the developer's own YouTube and TikTok accounts. It is not distributed to or used by any other person, and it does not offer sign-in, hosting, or any service to third parties.

What data the tool processes

The tool uses the YouTube Data API and the TikTok Content Posting API, authorized via OAuth, solely to upload video files and their titles, descriptions, and tags to the developer's own connected accounts. The OAuth access and refresh tokens issued to the tool are stored only in a local file on the developer's own computer. They are never transmitted to any server other than Google's or TikTok's own API endpoints, and never shared with any third party.

What data the tool does not do

  • It does not collect, store, or process data belonging to any other user or account.
  • It does not use any analytics, advertising, or tracking services.
  • It does not sell, rent, or share any data with third parties.
  • It does not read, message, or interact with other users' content on either platform.

Data retention

OAuth tokens remain on the developer's local machine until the developer revokes access (via Google Account settings or TikTok account settings) or deletes the local token file. No copy is kept anywhere else.
